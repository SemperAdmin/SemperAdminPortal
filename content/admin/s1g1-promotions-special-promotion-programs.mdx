## Role and responsibility

The S-1 officer owns special promotion program candidate routing per MCO P1400.32D<Citation source="MCO P1400.32D" /> Chapter 2 paragraph 2300 (Special Programs General), paragraph 2301 (PLC), paragraph 2302 (MEOP), paragraph 2303 (PSEP), paragraph 2304 (RPAP), paragraph 2305 (ECP), and paragraph 2306 (NROTC). The S-1 chief tracks program selection through the sponsoring authority. The promotions clerk maintains the unit special program roster. The commander signs the program endorsement on the candidate package.

Source documents flow from the candidate's eligibility documentation (academic transcripts, FITREPs, awards, PME), the program-specific application package, the chain-of-command endorsements, and the sponsoring authority release. Program actions land in MCTFS as program transition transactions. Outputs include the routed candidate package, the program selection record on release, the commissioning documentation when applicable, the Page 11 entry per MCO P1070.12K<Citation source="MCO P1070.12K" /> Chapter 6, and the unit special program roster entry.

Per MCO P1400.32D Chapter 2 paragraph 2300, special programs run alongside the standard enlisted promotion track. Each program targets a specific career path. Per Chapter 1 paragraph 1203 (Promotion Policies), the promotion authority shifts to the sponsoring authority on program selection.

## Trigger events

- Marine submits application to PLC, MEOP, PSEP, RPAP, ECP, or NROTC.
- Annual program board convenes for selection.
- Program selection released by the sponsoring authority.
- Candidate enters the program (commissioning, school, or training).
- Program disenrollment action received.
- Marine completes the program and commissions or graduates.

## Processing workflow

1. Identify the special program candidate. Pull the Marine's interest and the program-specific eligibility per Chapter 2 paragraphs 2301-2306.
2. Validate against the program-specific paragraph. PLC per paragraph 2301. MEOP per paragraph 2302. PSEP per paragraph 2303. RPAP per paragraph 2304. ECP per paragraph 2305. NROTC per paragraph 2306.
3. Route the candidate package to the program sponsor. The package routes through the chain to the sponsoring authority per the program rules.
4. Process program selection. Pull the selection release on publication. Match against the unit candidate roster.
5. Update MCTFS. Submit the program transition transaction through UD-MIPS per MCTFSPRIUM<Citation source="MCTFSPRIUM" />.
6. Coordinate commissioning or program entry. For commissioning programs (MEOP, ECP, NROTC), coordinate with the receiving command and the commissioning ceremony.
7. Process Page 11 documenting program selection. Per MCO P1070.12K<Citation source="MCO P1070.12K" /> Chapter 6 and MCO P1400.32D Chapter 1 paragraph 1205, document the program selection.
8. Update OMPF. Stage the program selection record in the OMPF Field Folder.
9. Brief the Marine. The S-1 chief briefs the Marine on the program path, the commissioning timeline, and the new obligations.
10. Maintain the unit special program roster. Track candidates by program, selection state, and projected entry date.

## Systems of record and forms

### Systems

- MCTFS. Holds program transition state.
- UD-MIPS. Cuts program transition transactions.
- MOL. Surfaces program selection and program-specific entries to the Marine.
- Program sponsoring authority (CMC MMOA-2, MMOA-3, OCS Command, NROTC Command). Issues selection and tracking.
- OMPF. Final record of program selection.

### Forms

- Program-Specific Application Package. Per the program rules.
- Chain-of-Command Endorsement Letter. Per Chapter 2 paragraphs 2301-2306.
- Selection Release. Issued by the program sponsoring authority.
- Commissioning Documentation. Per MCO 1400.31C<Citation source="MCO 1400.31C" /> for officer commissioning.
- NAVMC 118 (11) Administrative Remarks. Page 11 documenting program selection or disenrollment.

## Common pitfalls

- Program-specific paragraph misread. Per MCO P1400.32D<Citation source="MCO P1400.32D" /> Chapter 2 paragraphs 2301-2306, each program runs its own rules. Match the candidate to the correct paragraph.
- Chain-of-command endorsement skipped. The package routes without the unit endorsement. Route through the chain per the program rules.
- Program transition MCTFS transaction skipped. The pay and personnel system shows the Marine in original status. Cut the transition transaction per MCTFSPRIUM<Citation source="MCTFSPRIUM" />.
- Commissioning ceremony coordination dropped. The Marine commissions without the receiving command coordination. The S-1 officer coordinates per MCO 1400.31C<Citation source="MCO 1400.31C" />.
- Disenrollment action handled without Page 11. The OMPF lands without the disenrollment record. Cut the Page 11 per MCO P1070.12K<Citation source="MCO P1070.12K" /> Chapter 6.

## Decision points

- Commissioning versus non-commissioning special program. MEOP, ECP, NROTC, and PLC produce officers. PSEP and RPAP fill specific Marine Corps needs without commissioning. Per MCO P1400.32D<Citation source="MCO P1400.32D" /> Chapter 2 paragraphs 2301-2306, route per the program type.
- Active versus reserve program path. RPAP runs through the reserve component. Per paragraph 2304, the reserve path differs from the active commissioning programs.
- Program selection versus standard promotion track. The Marine sits in the standard composite zone and the special program candidate pool. Per Chapter 1 paragraph 1203, the program selection shifts the promotion authority to the sponsoring authority.
