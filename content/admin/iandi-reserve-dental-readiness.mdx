## Why Dental Readiness Matters

Per MCRAMM Chapter 6 and DODI 6025.19<Citation source="DODI 6025.19" />, dental status is a key component of deployment readiness. Members with significant dental issues does not deploy until treated. Dental issues also affect daily duty performance.

## Dental Classification System

Per DODI 6025.19<Citation source="DODI 6025.19" />, dental classifications.

### Class 1 (Optimal)

- No dental treatment needed
- Member fully dental-ready

### Class 2 (Routine Treatment Needed)

- Minor dental treatment required
- Treatment wait
- Member is dental-ready for deployment

### Class 3 (Treatment Needed Before Deployment)

- Significant dental issues
- Treatment required before deployment
- Member is NOT dental-ready

### Class 4 (Indeterminate)

- Dental status not assessed
- Does not determine readiness
- Member is NOT dental-ready

For deployment, member should be Class 1 or Class 2.

## Reserve Dental Care Access

Reserve members access dental care through.

### TRICARE Dental Program (TDP)

For SMCR members and families.

- Member-paid premium
- Civilian dental network
- Coverage for routine and major dental work

### Reserve Dental Care for Deployment

For pre-deployment dental work.

- Specific Reserve Dental Care program
- Might cover treatment for deployment readiness
- Specific eligibility per current policy

### Civilian Dental Insurance

Member's own civilian dental coverage if applicable.

### Active Duty Dental During Mobilization

When mobilized.

- Active duty dental coverage applies
- TRICARE Active Duty Dental Program

## Performance Steps for Dental Readiness Tracking

### Step 1 - Track Each Member

Maintain tracker.

- Last dental exam date
- Current dental classification
- Any treatment in progress

### Step 2 - Schedule Annual Dental Exams

For members.

- Annual dental exam
- Coordinate with TDP or other coverage
- Track to completion

### Step 3 - Address Class 3 Conditions

For members with significant dental issues.

- Coordinate treatment
- Reserve Dental Care if applicable
- Track until Class 1 or Class 2 achieved

### Step 4 - Pre-Mobilization Dental Screening

Before mobilization.

- Verify Class 1 or Class 2
- Coordinate any needed treatment
- Document for mobilization

### Step 5 - Update Records

After dental actions.

- Updated classification in MCTFS
- Documentation in member's record
- Coordinate with command

## Common Dental Issues

### Member Class 3 Without Treatment

Member has significant dental issues but not treated.

- Coordinate dental appointment
- Reserve Dental Care if applicable
- Track until resolved

### Member Without TDP Enrollment

Member has no dental coverage.

- Brief on TDP enrollment
- Coordinate civilian dental options
- Specific Reserve Dental Care for deployment

### Mobilization Delayed by Dental

Member alerted for mobilization but dental status blocks.

- Expedite treatment
- Specific procedures during pre-mobilization
- Might affect mobilization timing

### Member Refuses Dental Treatment

Rare but does occur.

- Specific procedures per MCRAMM
- Might affect continued drilling
- Coordinate with command

## TRICARE Dental Program (TDP)

For SMCR members and families.

### TDP Coverage

- Routine cleanings and exams
- Fillings and basic restorative
- Major dental work (crowns, root canals, etc.)
- Orthodontic coverage with limits
- Specific cost-sharing per service

### TDP Premium

- Member-paid monthly premium
- Family coverage available at higher premium
- Premium varies annually

### TDP Network

- Civilian dental network
- Network providers offer reduced rates
- Non-network use possible at higher cost

### TDP Enrollment

- Member enrolls via Beneficiary Web Enrollment (BWE)
- Annual enrollment cycles
- Specific changes during life events

## Coordination Touchpoints

Dental readiness touches.

- **TRICARE Dental Program**: For TDP coverage
- **Reserve Dental Care**: For specific reserve dental support
- **Member's civilian dentist**: For routine care
- **Local installation dental clinic**: For some Reserve dental services
- **Mobilization processing**: For pre-deployment dental

## Quality Control on Dental Readiness

Common errors.

1. **Tracker not current.** Member dental status not current. Maintain accurate tracker.
2. **Class 3 not addressed.** Member with deployment-blocking dental not getting treatment.
3. **TDP enrollment not coordinated.** Members without dental coverage.
4. **Pre-mobilization gap.** Member's dental found late in mobilization process.

## Documentation Required

For dental readiness.

- Dental exam documentation
- Dental classification
- Treatment records
- Updated MCTFS dental data

## Related Roles

Dental readiness affects every Reserve member's deployment eligibility.

- **Marines** maintain TDP enrollment, complete dental exams, address dental issues.
- **Leaders** track member dental status at the unit.
- **Commanders** at I&I oversee unit dental readiness posture.
