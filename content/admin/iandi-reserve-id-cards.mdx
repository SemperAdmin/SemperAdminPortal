## Reserve ID Card Categories

Per DODI 1000.13<Citation source="DODI 1000.13" /> and MCRAMM Chapter 5, Reserve members receive ID cards based on their component status.

### Drilling SMCR Member Card

For active drilling SMCR members.

- DD Form 2<Citation source="DD Form 2" /> (Reserve)
- Tan/beige color
- Lists Reserve component status
- Used for base access during drill weekends and AT
- Limited base access at non-drill times per installation policy

### Active Reserve (AR) Member Card

For full-time active duty reservists.

- DD Form 2<Citation source="DD Form 2" /> (Active)
- Same as active duty member card
- Full base access and benefits

### IRR Member Card

For IRR members.

- DD Form 2<Citation source="DD Form 2" /> (Reserve) with IRR designation
- Limited use for specific purposes
- Generally for VA and limited access

### Standby Reserve Member Card

For Standby Reserve members.

- DD Form 2<Citation source="DD Form 2" /> (Reserve) with Standby designation
- Limited use
- Specific purposes per program

### Retired Reserve Card

For Retired Reserve members.

- Different cards for awaiting pay vs receiving pay status
- Awaiting pay: Specific designation
- Receiving pay: Standard Retired card with full privileges

### Common Access Card (CAC)

For Reserve members performing duties requiring CAC access.

- Issued for specific drilling status
- Required for many systems access
- Specific renewal cycle

## Dependent ID Cards

Family members of Reserve members may receive ID cards.

### SMCR Member Dependents

- Eligible for dependent ID cards
- Limited access during drill weekends
- TRICARE Reserve Select coverage if enrolled

### Active Reserve Dependents

- Same as active duty dependent ID
- Full benefits

### Retired Reserve Dependents

- Receive dependent ID per retiree status
- Full base access for receiving-pay retiree dependents

## Performance Steps for ID Card Coordination

When member needs ID card.

### Step 1 - Verify Eligibility

Pull member's MCTFS record.

- Verify component status
- Verify rank
- Verify dependent enrollment if applicable

### Step 2 - Coordinate RAPIDS Appointment

Member must visit RAPIDS facility.

- Member schedules appointment via RAPIDS website
- Required documents identified
- Member appears in person

### Step 3 - Required Documentation

Member brings.

- Two forms of valid ID (one with photo)
- DEERS verification
- Specific component documentation
- Existing ID card to be replaced if applicable

### Step 4 - Card Issuance

At RAPIDS appointment.

- Verification of identity
- DEERS confirmation
- Photo taken
- Card issued

### Step 5 - DEERS Update

After card issuance.

- DEERS reflects new card
- Card details accessible to military systems
- Member can use card for base access

## Common ID Card Issues

These hit at the I&I shop.

### New SMCR Member Without ID

Newly arrived SMCR member without Reserve ID.

- Coordinate with RAPIDS for issuance
- Required during initial member receipt
- Provide member with appointment information

### Dependent Not in DEERS

Dependent without DEERS enrollment.

- Member must enroll via NAVMC 10922<Citation source="NAVMC 10922" /> first
- Then dependent gets DEERS enrollment
- Then dependent ID card

### ID Card Lost or Damaged

Member needs replacement.

- Police report for lost ID card
- RAPIDS appointment for replacement
- May incur fee for replacement

### ID Card Expiration

Member's ID card expired.

- Schedule renewal at RAPIDS
- May not access base without valid ID
- Coordinate timely renewal

### Status Change Requiring New Card

Member transitions between components.

- New ID card required for new status
- Coordinate with RAPIDS
- Old card returned

## Performance Steps for Dependent ID Coordination

For new dependent.

### Step 1 - Verify Dependent Status

Pull MCTFS dependent record.

- Verify NAVMC 10922<Citation source="NAVMC 10922" /> processed
- Verify dependent in DEERS
- Verify eligibility for ID card

### Step 2 - Coordinate RAPIDS Appointment

For dependent.

- Member or dependent schedules appointment
- Required documents identified
- Dependent appears in person

### Step 3 - Required Documentation

Dependent brings.

- Two forms of valid ID
- Birth certificate (for child dependents)
- Marriage certificate (for spouse)
- Member's ID for verification

### Step 4 - Card Issuance

ID card issued at appointment.

- DEERS verification
- Photo taken
- Card issued

## Common Dependent ID Issues

### Spouse Without Documentation

Newly married member's spouse without proper documentation.

- Marriage certificate required
- Spouse SSN required
- Coordinate documentation submission

### Child Without SSN

Newly born child without SSN.

- SSN application receipt may temporarily satisfy
- Permanent SSN required for full ID
- Coordinate with member

### Adult Disabled Child

Adult disabled child requiring continued ID.

- Specific medical documentation required
- Annual recertification typically
- Coordinate with member

### Divorce Removing Dependent

Former spouse loses dependent status.

- ID card returned at divorce
- DEERS removal processed
- Specific 20/20/20 rule for some long-term spouses

## ID Card and Base Access

ID cards drive base access privileges.

### Drilling SMCR Members

- Access during drill weekends and AT
- Specific access per installation policy at non-drill times
- Specific access for unit functions

### Family Members

- Access to commissary, exchange, MWR per current policy
- Specific access varying by installation
- Specific limitations for non-drilling member dependents

### Retired Reserve

- Full base access for receiving-pay retirees
- Limited access for awaiting-pay retirees per current policy

## Quality Control on ID Card Coordination

Common errors.

1. **Member arrives at RAPIDS without documentation.** Brief required documents in advance.
2. **DEERS not updated before RAPIDS visit.** Member receives ID but DEERS shows old data.
3. **Wrong card category issued.** Verify correct category before RAPIDS appointment.
4. **Dependent enrollment missed.** Process NAVMC 10922<Citation source="NAVMC 10922" /> before dependent ID coordination.

## Coordination Touchpoints

ID card processing touches.

- **RAPIDS facility**: For card issuance
- **DEERS**: For eligibility verification
- **Local installation pass and ID**: For specific access questions

## Related Roles

ID cards affect every Reserve member and their family.

- **Marines** maintain valid ID cards and ensure dependents enrolled.
- **Leaders** counsel members on ID card requirements.
- **Commanders** at I&I coordinate ID card services for unit members.
