NIS is the access manager for Marine Online applications, including UD-MIPS and DTMS. Log in to set up your requestor profile with the organization address, billet description, and Information Assurance training date, then select the correct supervisor. The supervisor approves the profile before any role request. Open Manage Access, then Manage My Access, search your ARUC, and submit. UD-MIPS routes Requestor to Supervisor to Information Owner approver. DTMS adds a Security Manager step. After approval, set the user from pending to active in UD/MIPS Manage Users. See the S-1 Admin Systems Management pages for the full workflow.
