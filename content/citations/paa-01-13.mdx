## Scope

PAA 01-13 sets administrative procedures for tracking and reporting Marines participating in DoD support to Hurricane Sandy relief efforts. The advisory assigns Crisis Code 9HS, personnel reporting instructions, and Special Interest Code SL0 for fiscal tracking of relief operations. Effective 27 October 2012 for active duty personnel; reserve component uses retro crisis code upon completion of active duty periods.

## Audience

- G-1 and S-1 staff tracking deployed personnel.
- IPAC and servicing PAC personnel.
- 0102, 0111, and 0170 admin specialists.
- Finance teams applying SIC SL0 to relief operation transactions.

## Content scope

- Assignment of Crisis Code 9HS for DoD support to Hurricane Sandy relief efforts.
- Personnel reporting instructions including hospitalization, individual location code, and TTC entries.
- TTC 801 000 COMPL ACDUTRA with type code D3 for reserve component activations supporting the relief effort.
- TTC 887 003 RETRO CRISIS CODE entry for reserve component upon completion of active duty.
- Assignment of Special Interest Code (SIC) SL0 to track expenditures from relief operations.
