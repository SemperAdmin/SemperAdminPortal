## Scope

PAA 01-13 sets administrative procedures for tracking and reporting Marines participating in DoD support to Hurricane Sandy relief efforts. The advisory assigns Crisis Code 9HS, personnel reporting instructions, and Special Interest Code SL0 for fiscal tracking of relief operations. The crisis code is effective 2012-10-27 for active duty personnel. The reserve component uses the retro crisis code entry upon completion of active duty periods.

## Audience

- G-1 and S-1 staff tracking deployed personnel.
- IPAC and servicing PAC personnel.
- 0102, 0111, and 0170 admin specialists.
- Finance teams applying SIC SL0 to relief operation transactions.

## Action language

- Assign Crisis Code 9HS to personnel supporting DoD Hurricane Sandy relief efforts, effective 2012-10-27 for active duty.
- Report personnel status per the advisory's instructions, including hospitalization, individual location code, and TTC entries.
- Report TTC 801 000 COMPL ACDUTRA with type code D3 for reserve component activations supporting the relief effort.
- Report TTC 887 003 RETRO CRISIS CODE for reserve component members upon completion of the active duty period.
- Apply Special Interest Code (SIC) SL0 to track expenditures from relief operations.

## Connection to Marine Corps policy

- The advisory follows the standing crisis code and special interest code pattern for named domestic support operations, pairing a personnel tracking code with a fiscal tracking code.
- The portal's I&I demobilization processing reference, sourced to MCRAMM Chapter 10 (MCO 1001R.1L), lists this advisory alongside PAA 01-11 and PAA 02-12 as deployment reporting authorities.
- Reserve activations under the relief effort close out through the COMPL ACDUTRA and retro crisis code entries in MCTFS.
