## Scope

The Marine Corps Total Force System Personnel Reporting Instructions Users Manual is the authoritative reporting-instructions reference for MCTFS users across the Marine Corps. Use it to validate transaction formats, code values, and reporting routes for personnel and pay events.

## Audience

S-1, IPAC, and admin specialists working inside MCTFS. Pay clerks, records clerks, and adjutants reach for it on every transaction edit and format question.

## Access notes

No public marines.mil URL points to the current manual. Pull the working copy from MOL or the M&RA portal. The next revision tracks the MCTFS release cadence, not a calendar date. Refresh the verified date in this record on a deliberate access of the live manual.
