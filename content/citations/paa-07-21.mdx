## Scope

This advisory addresses Separation Program Designator Codes For COVID-19 Vaccination Refusal. Issued by Headquarters Marine Corps Manpower and Reserve Affairs, Manpower Information Branch (MIF) and distributed through the MISSA portal for execution by S-1, IPAC, and commanders across the force.

## Audience

- S-1 desks and IPAC pay clerks executing the actions in this advisory.
- 0102, 0111, and 0170 admin specialists.
- Commanders signing affected actions.

## Operative paragraphs from the source

1. References (a) and (b) remain in effect. Request addressees disseminate this PAA to subordinate commands.
2. This advisory delineates administrative procedures for reporting the Separation Program Designator (SPD) Code for Marines who have refused the COVID-19 vaccination and are being processed for administrative separation.
3. If a Marine’s only basis for separation is refusing the vaccine, one of the following SPD codes are to be recorded in block 28 of the DD Form 214.
4. If another basis for separation exists (e.g. drug abuse) and the Marine is also being separated for vaccine refusal, enter the SPD code that is appropriate for the basis for separation, or contact HQMC (MMSR) for SPD code guidance.
5. This message applies to the Total Force, which includes recruits undergoing recruit training. For those in recruit training if vaccination refusal is the only basis, they will continue to be processed per PARA: 6205 for entry-level performance and conduct with an uncharacterized discharge.
