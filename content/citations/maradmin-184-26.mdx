## Scope

MARADMIN 184/26 announces the Navy Flight Demonstration Squadron (Blue Angels) is accepting enlisted Marine applications. The message supports the FY26 Blue Angels selection board listed in MARADMIN 224/26. Applicability is the Active Component.

## Audience

- Active Component enlisted Marines seeking Blue Angels assignment. Apply per the solicitation criteria.
- Career planners. Process applications through the unit Career Counselor.
- HQMC Manpower Management Enlisted Assignments (MMEA). Coordinates with the Blue Angels for selection.

## Action items

- Pull the application package requirements through the unit Career Counselor.
- Confirm obligated service through TFRS.
- Submit through the MOL EPAR module.

## Board timeline

MARADMIN 224/26, the CY2026 SNCO selection board schedule, lists the FY26 Navy Flight Demonstration Squadron (Blue Angels) board with an application window of 2026-11 and a convene date set by this message. The portal's SNCO selection boards page carries the same timeline.

## Connection to Marine Corps doctrine

- MARADMIN 224/26. CY2026 SNCO Selection Boards. Lists this message as the application solicitation and sets the 2026-11 application window.
- Marine Online (MOL) EPAR module. Application submission channel.
- Total Force Retention System (TFRS). Obligated service verification.

## Status

Signed 2026-04-23. Originator CMC Washington DC MRA MM. Applicability is the Active Component. Active for the FY26 Blue Angels selection cycle.
