## Source text

### Section 1

This transaction is used to report an absence In Hands of Civilian Authority (IHCA) that has been excused.  It is not valid for Manpower Information System Support Office input.  (See Chapter 7, Section 1 for chargeable leave reporting).

```
TTC 314 000 [A] IHCA [B] TO [C] ABS EXCUSED |
- [A] DOA (Date of Action) (8-bytes, Word Type
```
A)

- [B] FR DATE (From Date) (4-bytes, Word Type 4)
- [C] TO DATE (To Custody Date) (8-bytes, Word Type A)
