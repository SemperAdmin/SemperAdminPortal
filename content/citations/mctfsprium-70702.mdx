### Section 2

An advisory will be generated for members within the final month of a restricted promotion status period to remind Commanding Officers that the restricted promotion period will expire within 30 days. The new promotion restriction status codes will prevent the select pay grade advisory for the ranks of Private through Corporal for the restricted period.  If warranted, TTC 053 003 (PROM-RESTR-TERM) will be reported to terminate promotion restriction and reinstate eligibility for promotion prior to the system generated termination date.  This transaction will restore the promotion restriction code to a non-restricted status, and when the member is eligible, a select pay grade will be generated.

### Section 3

If a member is already in a restricted status, and has another administrative/punitive action during this time, and the restriction termination date for the second action is less than the first restriction termination date, the first restriction period and code will remain valid.  For example, reporting a SCM AWD 20010401 establishes a restriction status code of '2' and a termination date of 20010930.  The member then receives non-judicial punishment and the transaction NJP AWD 20010501 is reported.  Since the non-judicial punishment restriction period does not exceed the summary court-martial restriction, MCTFS will keep the restriction code and period for the summary court-martial.  Consequently, if the NJP AWD is 20010901, a restriction code of '1' (3 months) will be established with a termination date of 20011130 since it exceeds the summary court-martial restriction period.
