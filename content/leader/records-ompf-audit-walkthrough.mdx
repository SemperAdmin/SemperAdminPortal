## What OMPF Contains

Official Military Personnel File on Marine Online holds.

- FitRep history.
- Awards entries.
- Page 11 entries (commendatory and adverse).
- PME completion records.
- Training certifications.
- DD Form 214<Citation source="DD Form 214" /> components in build.
- Discipline history.

Promotion boards, separation processing, and post-service VA processing all read OMPF. An award not in OMPF is an award the system does not see.

## The Annual Section Audit

Schedule a section-wide OMPF audit annually. Build it into the section calendar.

- Each Marine pulls their own OMPF on Marine Online.
- Each Marine prints a snapshot.
- Section leader reviews each snapshot against unit records.
- Discrepancies flagged for S-1.

The Marine owns the self-verification. You own the cross-check.

## What to Verify

For each Marine's OMPF.

- FitRep history. Every annual report from EAS minus career back to enlistment present. No gaps.
- Award history. Every approved award present with correct date and citation.
- Page 11 entries. Both commendatory and adverse where they belong.
- PME completion. MarineNet courses, MCI, distance learning, school completion.
- Specific qualifications. Marksmanship, swim qual, instructor codes.
- Promotion warrants. Each promotion documented.

Missing entries fall into three buckets. Routing gaps (entry never made it). Transcription errors (entry made wrong). Source gaps (action never happened to be entered).

## What a Pre-Promotion Board Audit Looks Like

For Marines hitting the promotion zone, run an audit 6 months before the board.

- Pull OMPF.
- Pull FitRep history with attention to most recent reports.
- Pull award history.
- Cross-check unit records against OMPF.
- Submit corrections with enough lead time to clear before the board cut-off.

Boards see what is in OMPF on the board read date. Late corrections do not catch up.

## Submitting Corrections

For each discrepancy.

- Document the missing or incorrect entry.
- Pull the source document. Approved citation, signed FitRep, dated Page 11.
- Submit correction request through S-1 to the appropriate Branch.
- Track to closure.

Common correction destinations.

- FitReps. PEM Branch (MMRP).
- Awards. Awards Branch (MMPB-3).
- Page 11s and admin entries. Per the type.

S-1 routes the correction. The Marine and the section verify the result.

## Common OMPF Failures

- Audit skipped, gaps surface at the board.
- Correction submitted without source document. Returns.
- Marine never engaged in self-verification.
- Section keeps unit records, never reconciles to OMPF.
- New section leader inherits a Marine and audits never run before.
