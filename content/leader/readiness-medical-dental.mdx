## What MRC Means

Medical Readiness Class summarizes the Marine's deployability across medical conditions, immunizations, lab results, dental status, and required appointments.

| Class | Meaning |
|---|---|
| 1 | Fully ready, no action required |
| 2 | Partially ready, minor action needed |
| 3 | Not ready, deployment limiting condition |
| 4 | Status unknown or overdue assessment |

A section running on Class 1 and 2 is a section that can deploy on alert. A section with Class 3 and 4 Marines starts behind.

## Monthly Section Pull

Pull MRC for every Marine on a fixed monthly date.

- Identify Class 3 Marines. These are deploy-blockers.
- Identify Class 4 Marines. These are overdue or assessment gaps.
- Identify Class 2 Marines with appointments scheduled. Track to closure.
- Identify the action item for each below-standard Marine.

Common actions.

- Schedule the PHA (annual).
- Schedule dental cleaning or treatment.
- Update immunizations.
- Coordinate medical appointment for the limiting condition.
- Coordinate lab work.

## Dental Class Discipline

Dental Class drives a sub-component of MRC.

| Class | Meaning |
|---|---|
| 1 | No treatment needed |
| 2 | Minor treatment, deployable |
| 3 | Treatment required, deploy-limiting |
| 4 | Status unknown |

A Marine in Class 3 dental cannot deploy until treatment completes. Treatment can take weeks. The section catches Class 3 monthly, not at the alert.

## Reserve Component Considerations

Reserve Marines schedule their own medical and dental through Reserve Health Readiness or coordinate with civilian providers.

- Brief reserve Marines on the requirements.
- Coordinate Reserve Health Readiness Program enrollment.
- Track civilian provider coordination if the Marine pursues it.
- Verify the status update reaches the readiness system.

The leader cannot drive the appointment scheduling for reserve Marines the same way as active. The leader can drive the awareness and the follow-up.

## Tracking the Section

Maintain a section readiness sheet.

| Marine | MRC | Dental | Last PHA | Action | Due Date | Status |
|---|---|---|---|---|---|---|
| ... | ... | ... | ... | ... | ... | ... |

Review weekly. A Class 3 Marine sitting at Class 3 for two months is a Marine the section is not pushing.

## Briefing the Commander

Monthly section MRC and Dental status brief.

- Total Marines.
- Class 1 and 2 count.
- Class 3 count with action plan.
- Class 4 count with appointment scheduled.
- Trend over the prior 90 days.

Commander wants the trend, not just the snapshot.

## When the Deployment Alert Drops

A deployed-tomorrow alert with Class 3 Marines is a section in trouble.

- Some Class 3 conditions resolve fast (cleaning, fluoride). Schedule today.
- Some Class 3 conditions take weeks (extractions, implants). The Marine likely cannot deploy.
- Some Class 3 conditions require waivers. Coordinate with medical.

The fix is to never let Class 3 sit unaddressed in the first place.

## Common Tracking Failures

- Monthly pull skipped during a busy month.
- Class 4 Marines ignored because "they will get to it."
- Reserve Marines fall through the cracks.
- Treatment tracked but post-appointment update never verified.
- No commander brief, so the chain finds out at the alert.
