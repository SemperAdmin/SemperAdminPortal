## Use case

Run the Slate Report when you need a single view of Marine manpower across three states at once.

- On Had. Marines currently assigned to the command.
- Inbound. Marines projected to arrive within a date range.
- Outbound. Marines projected to depart within a date range.

## Run cadence

Non-routine. Pull it on demand for monitor briefs, turnover, or check-in and check-out planning.

## Audience

Enterprise. Run it with Enterprise Reports access.
